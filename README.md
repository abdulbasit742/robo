# Robo

This repository is prepared for a small educational robot arm project.

## Planned code file

`arduino_robot_arm_safe_controller.ino`

The controller should include:

- safe startup home position
- emergency stop command
- slow servo movement
- serial commands for base, shoulder, elbow, and gripper
- angle limits for each joint

## Hardware notes

Use a separate 5V power supply for servos and connect all grounds together.
Keep hands away from moving joints while testing.
